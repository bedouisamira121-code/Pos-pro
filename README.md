# POS Pro — Vendeur Mobile

## 🚀 Déploiement sur GitHub Pages (PWA)

### Fichiers à uploader :
- `index.html` (renommer POS_Pro_PWA.html)
- `manifest.json`
- `sw.js`
- `icon-192.png` (optionnel)
- `icon-512.png` (optionnel)

### Étapes :
1. Créez un compte sur [github.com](https://github.com)
2. Nouveau dépôt : **pos-pro** (Public)
3. Uploadez tous les fichiers
4. Settings → Pages → Source: **main branch** → Save
5. Votre URL : `https://VOTRE_NOM.github.io/pos-pro`

### Sur Android :
1. Ouvrez l'URL dans **Chrome**
2. Une bannière **"Installer POS Pro"** apparaît automatiquement
3. Appuyez sur **Installer** → Application sur l'écran d'accueil ✅

### Sur iPhone (Safari) :
1. Ouvrez l'URL dans **Safari**
2. Appuyez sur **Partager** (carré avec flèche)
3. **"Sur l'écran d'accueil"** → Ajouter ✅

### Fonctionnalités PWA :
- ✅ Fonctionne **hors ligne** (Service Worker)
- ✅ Icône sur l'écran d'accueil
- ✅ Plein écran (sans barre de navigateur)
- ✅ Données sauvegardées localement
- ✅ Backup JSON + Google Drive
